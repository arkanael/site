Sobre o curso

O curso de Programação é voltado para quem quer entrar no mercado de tecnologia. Você aprenderá desde a lógica de programação até o desenvolvimento de aplicações web completas.

Nossa metodologia é hands-on, com projetos práticos que simulam situações reais do mercado de trabalho. Ao final, você terá um portfólio completo.

O que você vai aprender

Módulo 1: Lógica de Programação

6 semanas
  • Algoritmos e fluxogramas
  • Variáveis, tipos de dados e operadores
  • Estruturas condicionais e de repetição
  • Funções e procedimentos

Módulo 2: Frontend Básico

8 semanas
  • HTML5 semântico e estruturação
  • CSS3, Flexbox e Grid Layout
  • JavaScript básico e DOM
  • Responsividade e Mobile First

Módulo 3: JavaScript Avançado

6 semanas
  • ES6+ e programação funcional
  • APIs e requisições HTTP
  • Manipulação de dados (JSON)
  • Frameworks (React básico)

Módulo 4: Projeto Final

4 semanas
  • Planejamento e arquitetura
  • Desenvolvimento de aplicação completa
  • Versionamento com Git/GitHub
  • Deploy e apresentação
